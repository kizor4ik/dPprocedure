# dPprocedure
 рабочий код для vrch
Три разных файла под разные режимы.
Каждый файл разделен на три блока которые нужно проентерить:
1) Материальные параметры траектории, холестерика
2) Блок инициализации, тут записаны все формулы и обозначения из файла vrch
3) Блок вывода данных
# Вывод данных
Имеются 2 процедуры в каждом файле TheoryCheck[s,m,ko,np] и GetProbability[s,m,ko,np]. На вход принимают квантовые числа закрученных фотонов в таком порядке. 
Первый метод возвращает выполнение условия применимости данной теории в данной области параметров (оценка остается пользователю).
Второй метод возвращает вероятность.
